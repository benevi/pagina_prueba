HTML y CSS

Idiomas utilizados para construir y diseñar las apariencias de sus sitios web. HTML agrega la estructura y CSS la estiliza.

jQuery

Una biblioteca de JavaScript rápida, rica en funciones y fácil de usar que maneja animaciones y solicitudes de Ajax.

Ruby

Un lenguaje dinámico de propósito general que es simple y productivo. Se usa comúnmente en todo el mundo para crear aplicaciones web.

PHP

Un lenguaje de script de código abierto que puede integrarse en HTML y ser adecuado para el desarrollo web.
